kijiji的一点小感想

Oct 28 2017

在国内的时候从没接触过二手交易，觉得二手平台的水很深，这可能是自己的偏见，但我关注的是大件物品如家具的二手买卖，这个和租房市场很有关系，国内没有加拿大这样成熟的出租公寓，大部分是个人出租，家具都有提供，租客就是拎包入住，但是加拿大这边公寓不提供家具，搬出去的人要处理家具，搬进去的人要找个便宜的家具，造成二手交易的繁荣。

刚到加拿大的时候，觉得新家具有点贵，舍不得买，发现中文论坛上有二手家具交易，但当时只是猎奇的心理每天刷刷看，就算是看上某件东西也是纠结着和对方咨询，迟迟不敢下手，有点不放心。

再后来发现了kijiji这个不得了的网站，可我连中国人自己的交易平台都不敢尝试，更何况老外的平台？一直到和老婆要搬到一起住的时候，她提议把新家的家具重点放在kijiji上，我那时候也是带着一种不靠谱的心态听从，其实自己没有怎么上心……老婆就每天刷kijiji，我们再一起去看家具，交易，搬回来……就这样我们买了不少二手家具，价格都很便宜，几十块钱，从桌子、椅子到柜子。而且他们都在质量和外观上很不错。这是我第一次尝到甜头，而且发现和老外交易也没有想象中的那样困难，看来抽象的感觉止步于体验。

从伦敦搬到休斯顿是个非常突然的计划，需要把所有的家具处理掉，而且时间只有一个多月。当时在一月份，非常冷，还有大雪。我们决定把大部分东西都在kijiji上处理，同时还可以获得quick money，用于搬家折腾。我记得当时先从两个巨大的游泳圈下手，觉得它们肯定要卖掉……没想到下午放到网上，傍晚就来了中国人把它们买走了，当时可是下着大雪，太搞笑了……接着就是沙发，桌子，椅子，床……当时卖这些东西很有挑战，因为有的东西我们还需要用，所以买家来的时候需要商量取货的时间，还有押金的问题……期间我卖的起劲，把小东西也顺带卖了，比如自己的手机、耳机、镜头等。直到最后走的一天，买床的人把我们的床取走。现在想想，真是太不简单了，当时自己真是full-time卖东西，如果没有那么多精力，不太可能办到。最开始的时候我们还是打算把房子整体转租，这样所有的家具都可以卖给一个人，方便而且双赢。这样看起来省时省力，但操作起来还是发现可能性很小，看客是有，但最后都没能成。

再到最近，和老婆在公园散步的时候突发奇想，要改造客厅，让我们保持对现在公寓的新鲜感，cover房子太小的硬伤，重新爱上它……要卖掉沙发和电视。我是心急派，立马拍照po到kijiji上，然后就是是不是关注下ad状态，期待email被轰炸……沙发最初的价格是550，没人问，很快就沉了，发现kijiji上卖沙发的很多，而且有不少dealers，这也说明沙发比较难卖，因为不好搬……后来把价格调到400，还是很少有人关注……心急之下把价格放到200，真是吐血的价格了，当时也是好奇一个问题：究竟是沙发供远大于求没人买还是我的价格没有吸引力。后来感兴趣的人猛增证明了低价格肯定是有吸引力的，而且有人是在kijiji上一直关注好deal，及时下手。最后一对夫妇下午询问，晚上搬走，整个过程不到一周。奇迹。

之前觉得kijiji有点low，大部分人应该都是在需要某件东西但又budget不够的情况下才考虑这个网站。但不是这样，有不少人一直持续关注kijiji的东西，只要发现特别给力的东西，就会下手，就算那个东西不是他们的刚需。而且这些买家其实生活很decent。所以当你急需抛售东西的时候，标很低的价格绝对会对你有帮助。

在加拿大kjiji交易的人大部分都很靠谱诚信，我在卖GPS的时候，没有自己事先检查，在交易过程中一直没信号，当时很囧，就想拿回GPS不卖了，最后那个买家跟我也只是抱怨他用过这个牌子，知道有这种情况，然后还问我想多少钱卖，我说你随便给个钱就行……而且做好了20刀的心理预期，结果他就还了10刀（我标的价格是50刀）。还有卖Apple TV的时候买家也盒子都没打开看就给钱走了……他们真的是想买东西并且相信你。在这真的要感谢加拿大这样简单诚信的社会！

在kijiji上卖小东西，而且价格给力（也不是很低），肯定会很抢手（比如apple tv、coffee table），基本上是多人同时要来看，并且当天可以结束交易。

会有不少ghost买家，用kijiji default message咨询你的东西还在不在，但之后就没有回音了，他们不是serious买家，只是好奇一下，所以不要把时间放在邮件回复上，对于所有咨询都是简单回复：还在，要不要来看下。没必要在邮件里详细推销或者追问。重点是让他们先来看一下，可以不用立刻买。因为亲眼看过才会让他们产生这是好货冲动要买的心态（前提你卖的就是好货），而且他们既然都跑过来看了，不想再约时间付钱了，就会立刻买下。我在卖casio电子琴的时候，买家就是只想看看，结果发现确实很好，想尽各种办法也要买下。

一种passive的卖东西模式就是避免在邮件沟通，让买家过来看，犹豫的买家会不停邮件询问，serious买家会同意先来看一下。在邮件交流上拉长时间并不会促成买卖。消极等待买家就好。
